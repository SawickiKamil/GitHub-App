# GitHub Api App

Simple application which fetching data from GitHub API

### Tech

- [React.js] - hooks!
- [Material-UI]
- [TypeScript]
- [CSS in JS] - JSS
- [Axios]

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd GitHub-App
$ npm install -d
$ npm run start
```
